# vue-shop

### 项目配置（初始化项目安装依赖）
```
npm install
```

### 编译启动项目
```
npm run serve
```

### 编译打包项目
```
npm run build
```

### 运用的技术栈
```
vue.js    
vue router      管理路由   
Vue CLI         快速创建项目  
vuex            管理共享的数据状态  
axios           请求库  
element         一个第三方组件库   
echarts         视图     
```
